# Setup & Installation — TONI 2.0

Dieses Dokument beschreibt die Installation, den Start und die grundlegende Projektstruktur von TONI 2.0.

---

## 1. Voraussetzungen

- Node.js (empfohlen: aktuelle LTS-Version)
- Ein Terminal (macOS: Terminal oder iTerm)
- Ein Editor (z. B. VS Code)
- Git (optional, aber empfohlen)

---

## 2. Projekt installieren

```bash
git clone <repo-url>
cd toni
npm install
