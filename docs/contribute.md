# Contribution Guide — TONI 2.0

Dieses Dokument beschreibt die Regeln und Abläufe für Mitarbeit an TONI 2.0.

---

## 1. Grundprinzipien

- Klare, nachvollziehbare Struktur
- Saubere, kleine Commits
- Keine Logik in UI-Komponenten
- Module strikt getrennt halten
- Dokumentation aktuell halten
